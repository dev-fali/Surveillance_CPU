## Get started
1. Install python 
2. Create a virtual environment (optional)
```bash
   python -m venv venv
   ```
3. Active the virtual environment
On windows:
```bash
   .\venv\Scripts\activate
   ```
On MacOs/Linux:
```bash
   source venv/bin/activate
   ```
4. Install required libraires
```bash
   pip install psutil slack_sdk python-dotenv matplotlib
   ```

5. SMTP Ports
   Gmail
